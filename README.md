# Agri Store - Agriculture E-commerce

A modern, premium e-commerce application for agricultural products.

## Project Setup

The project is built with React, Vite, and Tailwind CSS.
Due to some environment restrictions, the dependencies might not be fully installed.

### How to Run

1.  Open your terminal in this directory:
    ```bash
    cd C:\Users\prave\.gemini\antigravity\scratch\agri-store
    ```

2.  Install dependencies:
    ```bash
    npm install
    npm install -D tailwindcss postcss autoprefixer
    npm install lucide-react
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open the link shown in the terminal (usually http://localhost:5173).

## Features

- **Premium Design**: Custom "Outfit" typography and organic color palette.
- **Product Grid**: Browse seeds, tools, and fertilizers.
- **Cart Interaction**: Add items to cart (simple state tracking).
